Grid style properties

Initialize size of grid lines using grid-area:


.grid-container header {
  grid-area: header;
}

.grid-container main {
  grid-area: main;
}

.grid-container section {
  grid-area: section;
}

.grid-container aside {
  grid-area: sidebar;
}

.grid-container footer {
  grid-area: footer;
}

.grid-container {
  display: grid;
  grid-template-columns: 1fr 1fr 1fr 1fr;
  grid-template-rows: auto;
  grid-template-areas:
    "header header header header"
    "main main main main"
    "section section section sidebar"
    "footer footer footer footer";
}

Grid style properties

Setting gap between gridlines:


.grid-container {
  grid-column-gap: 10px;
  grid-row-gap: 15px;
}

A short hand property for grid gap:


.grid-container {
  grid-gap: 15px 10px;
}
