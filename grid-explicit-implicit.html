The difference between explicit and implicit grids

CSS grid layout uses the concept of an explicit grid and an implicit grid. This is a key concept that you need to be aware of when creating a grid, otherwise you could end up with a bunch of rows or columns that you didn't expect!

Explicit Grid

  • Define with the grid-template-rows, grid-template-columns, and grid-template-areas properties.
  • Uses grid-template-rows and grid-template-columns.

Implicit Grid

  • Automatically generated by the grid container whenever grid items are positioned outside of the explicit grid.
  • Uses grid-auto-rows and grid-auto-columns.

Explicit Grid Example

#grid {
  display: grid;
  grid-template-rows: 140px 140px;
  grid-template-columns: 1fr 1fr;
  grid-gap: 10px;
}
#grid > div {
  padding: .5em;
  background: #fff;
  text-align: center;
}

Implicit Grid Example

#grid {
  display: grid;
  grid-template-rows: 140px 140px;
  grid-template-columns: 1fr 1fr;
  grid-gap: 10px;
  grid-auto-rows: 140px;
}
#grid > div {
  padding: .5em;
  background: #fff;
  text-align: center;
}
