Fractional unit - 'fr'

Represents a fraction of the leftover space in the grid container.

1 Fr = 1 part of available space.

Code

.grid {
display: grid;
grid-template-columns: repeat(4, 25%);
grid-column-gap: 10px;
grid-template-rows: 100px;
margin: 0;
}
